# Simple Games

Небольшая Angular-песочница для простых браузерных игр.

Стек: Angular 21 standalone components + SVG.

## Roadmap

1. Pocket Pet v1: питомец появляется сразу ребенком, взрослеет по этапам `child -> teen -> adult` и всегда уходит: либо потому что вырос и начинает собственную жизнь, либо потому что ему не хватало заботы. Смерти и ожидания вылупления нет.
2. Farewell & Results: при уходе сохраняются причина, дата, итоговая фраза, финальный балл заботы, финальные статы и подробная история действий для таблицы выращенных питомцев в профиле.
3. Care Loop: базовая еда, джанкфуд как быстрый бонус с вредом здоровью, уборка, игра, прогулка с временной недоступностью питомца и свет как часть режима сна.
4. Species Traits Foundation: у каждого вида питомца есть коэффициенты реакции на действия и ухудшение статов со временем. Пока все коэффициенты равны `1`, но модель уже допускает разные предпочтения видов.
5. Activity Metadata Foundation: действия получают метаданные для будущих систем: цена в силах игрока, тип активности и теги восприятия питомца. Текущее поведение ухода остается прежним.
6. Player Energy & Activity Budget: у игрока появляются силы для активностей; часть действий бесплатна, часть тратит ресурс, а ресурс восстанавливается со временем.
7. Pet Perception v1: питомец формирует мнение об игроке по истории заботы: строгость, балование, регулярность заботы и невнимательность.
8. Question Activity: питомец задает вопросы, а вопросы и варианты ответов берутся из базы.
9. Pet Activities: мини-игры позже расширяют прогулки, игры и другие активности, когда базовый цикл ухода стабилизируется.
10. Backend Foundation: отдельный NestJS API, PostgreSQL и Prisma после стабилизации игровых правил. Backend становится источником истины для состояния питомца, времени, действий и ресурсов игрока.
11. Account & Sync: авторизация, перенос localStorage-состояния в аккаунт и синхронизация между устройствами.
12. Realtime Pet Session: при открытой активной сессии питомца frontend получает обновления состояния через WebSocket/SSE.
13. Progression: достижения, предметы, новые питомцы, разблокировка дракона и новые мини-игры.

Текущий backend-вектор: NestJS + PostgreSQL + Prisma. Next.js API не выбран, потому что frontend остается Angular; Supabase можно рассматривать позже как managed Postgres/Auth альтернативу.

## Старт

```bash
npm install
npm start
```

По умолчанию dev-server слушает `0.0.0.0:4200`, поэтому приложение можно открыть с другого устройства в той же Wi-Fi сети.

На компьютере открой:

```text
http://localhost:4200
```

На телефоне или другом устройстве в той же сети открой:

```text
http://<IPv4-адрес-компьютера>:4200
```

IPv4-адрес можно посмотреть командой:

```powershell
ipconfig
```

Если нужен только локальный запуск на компьютере:

```bash
npm run start:local
```
